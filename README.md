## Proyectos Destacados

### [Sistema de Control Presupuestario — UEB](https://github.com/Jeffs4nchez/Sistema_Control_Presupuestario_UEB_G51_58_Backend)

> Sistema de gestión financiera institucional de escala real para la Universidad Estatal de Bolívar.

![Laravel](https://img.shields.io/badge/Laravel_12-FF2D20?style=flat-square&logo=laravel&logoColor=white)
![React](https://img.shields.io/badge/React_19-61DAFB?style=flat-square&logo=react&logoColor=black)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-4169E1?style=flat-square&logo=postgresql&logoColor=white)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-06B6D4?style=flat-square&logo=tailwindcss&logoColor=white)

- 17 historias de usuario: certificaciones, liquidaciones, reportes CSV, auditoría
- REST API con 11 controllers, 17 modelos Eloquent, Observers y comandos Artisan
- Control de acceso por roles (RBAC) con 5 perfiles diferenciados
- Notificaciones automáticas por email con plantillas HTML corporativas
- Patrones de diseño GOF: Observer, Template Method, Factory

[![Backend](https://img.shields.io/badge/Ver_Backend-FF2D20?style=for-the-badge&logo=laravel&logoColor=white)](https://github.com/Jeffs4nchez/Sistema_Control_Presupuestario_UEB_G51_58_Backend)
[![Frontend](https://img.shields.io/badge/Ver_Frontend-61DAFB?style=for-the-badge&logo=react&logoColor=black)](https://github.com/Jeffs4nchez/Sistema_Control_Presupuestario_UEB_G51_58_Frontend)
