## 🚀 Proyectos Destacados

### 🏛 [Sistema de Control Presupuestario — UEB](https://github.com/Jeffs4nchez/Sistema_Control_Presupuestario_UEB_G51_58_Backend)

> Sistema de gestión financiera institucional de escala real para la Universidad Estatal de Bolívar.

- **Stack:** Laravel 12 · React 19 · PostgreSQL · TailwindCSS · Framer Motion
- 17 historias de usuario implementadas — certificaciones, liquidaciones, reportes CSV, auditoría
- Arquitectura REST API con 11 controllers, 17 modelos Eloquent y sistema de Observers (GOF)
- RBAC con 5 roles diferenciados y ciclo de estados completo para documentos presupuestarios
- Notificaciones automáticas por email con plantillas HTML corporativas
- Usado por personal administrativo real de la UEB

[![Backend](https://img.shields.io/badge/Ver_Backend-FF2D20?style=flat-square&logo=laravel&logoColor=white)](https://github.com/Jeffs4nchez/Sistema_Control_Presupuestario_UEB_G51_58_Backend)
